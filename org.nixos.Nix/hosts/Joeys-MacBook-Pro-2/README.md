This is for configuring Nix on my MacBook Pro.

## [`configuration.nix`](configuration.nix)

This file installs some packages specific to my MacBook (including Homebrew and can even manage packages in Homebrew's respoitories but not Nix's) and configures some other miscellaneous things.
